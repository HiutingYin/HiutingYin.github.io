### 根据地址获取经纬度
代码示例：
```
    public function getLatAndLng($address)
    {
        $key = 'yourTencentMapKey';//申请的腾讯地图key
        $url = 'https://apis.map.qq.com/ws/geocoder/v1/?address=' . $address . '&key=' . $key;
        $info = file_get_contents($url);
        $info = json_decode($info, true);
        if ($info['status'] == 0) {
            return [
                'lat' => $info['result']['location']['lat'],//纬度
                'lng' => $info['result']['location']['lng']//经度
            ];
        } else {
            return false;
        }
    }
```
